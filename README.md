A lightweight web application designed to help students browse, filter, and compare universities based on key performance indicators stored in a database.

##  Project Overview
The goal of this project is to simplify the university selection process. It provides a clean interface to view academic data, tuition fees, and admission requirements, allowing for a side-by-side comparison of different institutions.

##  Features
* **Landing Page:** Introduction to the project goals and navigation.
* **University Catalog:** A structured list of universities fetched from the DBMS.
* **Comparison Tool:** Compare universities by entry scores, location, and costs.
* **Responsive Design:** Clean and minimal UI that works on both desktop and mobile devices.

##  Tech Stack
* **Frontend:** HTML5, CSS3 (Vanilla)
* **Backend (Planned):** PHP / Python (Flask/Django) or Node.js
* **Database:** MySQL / PostgreSQL
